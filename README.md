# tauri vite example

[tauri](https://github.com/tauri-apps/tauri) port of https://github.com/hi-ogawa/electron-vite-experiment

```sh
# development
pnpm i
pnpm dev

# build (see misc/pkgbuild/README.md for installation on archlinux)
pnpm build
```
